# 🎓 Student Management System

A web-based **Student Management System** built using **Django** that allows administrators to manage student records efficiently.

This project demonstrates CRUD operations, authentication, pagination, and role-based access using Django.

---

## 🚀 Features

✔ User Authentication (Login / Logout)  
✔ Role-Based Access (Admin vs Staff)  
✔ Add Student  
✔ Edit Student  
✔ Delete Student  
✔ Student List Dashboard  
✔ Pagination  
✔ Search Students  
✔ Clean UI using HTML & Bootstrap

## ⚙️ Installation

1. Clone the repository

git clone https://github.com/Ankusam1/student-management-system.git

2. Go into project folder

cd student-management-system

3. Run server

python manage.py runserver
